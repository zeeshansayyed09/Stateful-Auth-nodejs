# Stateful Authentication

A simple Node.js, Express, MongoDB project demonstrating manual stateful authentication.

## Features

- User Signup
- User Signin
- Password Hashing (bcrypt)
- Session-based Authentication
- Cookies
- Protected Routes
- Logout

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- bcrypt
- cookie-parser
